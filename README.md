# 🚀 Bootcamp Projects

Bu depo, programlamaya ve C# diline ilk adım attığım dönemde yazdığım temel düzeydeki kod çalışmalarını içermektedir. 

Kodlar başlangıç seviyesinde olsa da; algoritmalar, döngüler ve temel programlama mantığını oturttuğum, benim için oldukça önemli ve nostaljik bir arşivdir. Geriye dönüp gelişimimi görmek amacıyla bu repoyu public olarak saklıyorum.

## 🛠️ Teknolojiler
* C#
* .NET Console Applications

## 📂 Neler Var?
Bu repo genel olarak aşağıdaki konulardaki pratiklerimi ve ilk projelerimi içerir:
* Temel Veri Tipleri ve Değişkenler
* Kontrol Yapıları (If-Else, Switch-Case vb.)
* Döngüler (For, While, Foreach)
* Diziler ve Listeler
* Metotlar (Fonksiyonlar)
* Nesne Yönelimli Programlamaya (OOP) Giriş
